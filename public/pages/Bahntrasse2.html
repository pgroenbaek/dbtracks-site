[t00]NR_Bahntrasse 2 [t01]This package contains 4,279 shape models for route bulding in MSTS.
 
It provides various types of track sections with integrated path elements, embankments, ramps and retaining walls, exactly matching 1 to 4 track sections available in XTracks 3.20.
Using the special instructions for 'Track Shape Substitution', ('A'-) track sections can be easily replaced by these elements.
See also: NR_Bahntrasse_1  (static path elements)
 
The provided textures are examples only. They are subject to be replaced by own creations (even subareas). [t02]Download [t03](81.4 MB) [t04]Please click on a picture for an enlarged version [t05]
