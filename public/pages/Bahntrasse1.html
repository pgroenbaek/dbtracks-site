[t00]NR_Bahntrasse 1 [t01]This package contains 4,094 shape models for route bulding in MSTS.
 
It provides various types of path elements, bridges, embankments, ramps and retaining walls, exactly matching 1 to 4 ('A'-) track sections available in XTracks 3.20.
Using the special instructions for Position Adjustment, the static objects can be easily adjusted.
See also: NR_Bahntrasse_2  (track sections with integrated path elements)
